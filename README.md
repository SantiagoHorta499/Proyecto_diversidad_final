Se realizó una recolección de datos en Genbank y iNaturalis para la especie Rana temporaria, esto para responder la pregunta:
¿Difieren la diversidad genética (π) y la densidad de registros de observación de Rana temporaria entre las poblaciones del sur de Europa y las del norte,
y qué podría explicar estas diferencias?
Después de correr los códigos, se encontró que la diversidad genética de esta especie se diferencia basnte según la latitud donde se encuentre, y esto se puede
deber a procesos de colonización post glaciares que preservaron solo una parte del pool genético original, y obviamente debido al esfuerzo de muestreo para la 
especie.
